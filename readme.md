# 実行方法

## 実行環境
Python - 3.10.11

## git cloneする

```
$ git clone https://github.com/MiuraManato/Vue-simple-diary-Application.git
```
## 必要なpipファイルをインストールする
requirements.txtを使ってインストールできます
```
$ pip install requirements.txt
```
Pythonが複数のバージョン入っている場合は以下の通りです
```
$ py -3.10 -m pip install requirements.txt
```
